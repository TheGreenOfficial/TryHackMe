**Intorduction:** <br><br>
**HTTP (HyperText Transfer Protocol) is the standard protocol used to transfer web pages and data between your browser and a web server. And HTTPS (HyperText Transfer Protocol Secure) is the encrypted and secure version of HTTP that protects data and verifies the website’s identity.**
<br><br>
**Questions:** <br>
**1) What does HTTP stand for?** <br>
**-> HyperText Transfer Protocol**
<br><br>
**2) What does the S in HTTPS stand for?** <br>
**-> secure**
<br><br>

**Requests & Responses:** <br><br>
**A browser uses a URL to send an HTTP request to a web server, and the server returns an HTTP response containing the requested webpage or data.**
<br><br>
**Questions:** <br>
**1) What HTTP protocol is being used in the above example?** <br>
**-> HTTP/1.1**
<br><br>
**2) What response header tells the browser how much data to expect?** <br>
**-> Content-Length**

**HTTP Methods:** <br><br>
**HTTP methods define the action a client wants to perform on a server, such as GET (retrieve data), POST (send/create data), PUT (update data), and DELETE (remove data).**
<br><br>
**Questions:** <br>
**1) What method would be used to create a new user account?** <br>
**-> POST**
<br><br>
**2) What method would be used to update your email address?** <br>
**-> PUT**
<br><br>
**3) What method would be used to remove a picture you've uploaded to your account?** <br>
**-> DELETE**
<br><br>
**4) What method would be used to view a news article?** <br>
**-> GET**

**HTTP Status Codes:** <br><br>
**HTTP Status Codes: HTTP status codes are server responses that indicate the result of a client’s request, such as success (200), redirection (300s), client errors (400s), or server errors (500s).**
<br><br>
**Questions:** <br>
**1) What response code might you receive if you've created a new user or blog post article?** <br>
**-> 201**
<br><br>
**2) What response code might you receive if you've tried to access a page that doesn't exist?** <br>
**-> 404**
<br><br>
**3) What response code might you receive if the web server cannot access its database and the application crashes?** <br>
**-> 503**
<br><br>
**4) What response code might you receive if you try to edit your profile without logging in first?** <br>
**-> 401**

**Headers:** <br><br>
**HTTP headers are extra pieces of information sent between a client and server to control communication, data handling, caching, cookies, compression, and content types.**
<br><br>
**Questions:** <br>
**1) What header tells the web server what browser is being used?** <br>
**-> User-Agent**
<br><br>
**2) What header tells the browser what type of data is being returned?** <br>
**-> Content-Type**
<br><br>
**3) What header tells the web server which website is being requested?** <br>
**-> Host**

**Cookies:** <br><br>
**Cookies are small pieces of data stored in your browser that websites use to remember users, maintain sessions, and store settings across HTTP requests.**
<br><br>
**Questions:** <br>
**1) Which header is used to save cookies to your computer?** <br>
**-> Set-Cookie**
