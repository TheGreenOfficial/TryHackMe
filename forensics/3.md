### Task 4

**TrueCrypt**

- What is the TrueCrypt passphrase?

TrueCrypt is disk encryption software that encrypts files, partitions, or entire drives. Normally, if you only have the encrypted container, you need the correct password to decrypt it. when a user mounts a TrueCrypt volume, the passphrase and encryption keys may temporarily exist in RAM so the software can access the encrypted data.

```
# truecryptpassphrase - this plugin scans the Windows memory dump for cached TrueCrypt password structures.
skid@TheGreen:~/ctfs$ volatility -f Snapshot14.vmem truecryptpassphrase --profile Win7SP1x64
Volatility Foundation Volatility Framework 2.6.1
Found at 0xfffff8800512bee4 length 11: forgetmenot
skid@TheGreen:~/ctfs$
```

> forgetmenot
