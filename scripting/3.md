**Introduction** <br>
**Bash is a command-line scripting language used to automate tasks and control a system by executing sequences of commands in a terminal.**

**Our first bash scripts.** <br>
**A bash script starts with a shebang (#!/bin/bash), runs commands like echo, ls, or whoami, and must be made executable with chmod +x before running it with ./script.sh.**

<br>

**Questions** <br>
**What piece of code can we insert at the start of a line to comment out our code?** <br>
**Ans: #** <br><br>

**Parameters** <br>
**Bash parameters let you pass input into scripts using $1, $2, etc., or interactively collect input with read for dynamic user-driven execution.**

<br>

**Questions** <br>
**How can we get the number of arguments supplied to a script?** <br>
**Ans: $#** <br><br>
**How can we get the filename of our current script(aka our first argument)?** <br>
**Ans: $0** <br><br>
**How can we get the 4th argument supplied to the script?** <br>
**Ans: $4** <br><br>
**If a script asks us for input how can we direct our input into a variable called ‘test’ using “read”** <br>
**Ans: read test** <br><br>
**What will the output of “echo $1 $3” if the script was ran with “./script.sh hello hola aloha”** <br>
**Ans: hello aloha**

**Arrays** <br>
**Bash arrays store multiple values in one variable using indexed positions (starting at 0), allowing you to access, modify, or delete elements with syntax like ${array[index]} and unset.** <br>
**Given the array please answer the following questions: `cars=('honda' 'audi' 'bmw' 'tesla')`**

<br>

**Questions** <br>
**What would be the command to print audi to the screen using indexing.** <br>
**Ans: echo "${cars[1]}"** <br><br>
**If we wanted to remove tesla from the array how would we do so?** <br>
**Ans: unset cars[3]** <br><br>
**How could we insert a new value called toyota to replace tesla?** <br>
**Ans: cars[3]='toyota'**

**Conditions** <br>
**Bash conditionals use if [ condition ] ... fi with comparison operators (like -eq, -gt) to execute code based on conditions, including checking values, user input, or file properties.**

<br>

**Questions** <br>
**What is the flag to check if we have read access to a file?** <br>
**Ans: -r** <br><br>
**What is the flag to check to see if it's a directory?** <br>
**Ans: -d**
